To find a list of the individual’s collaborators, known as “pi”, you add in “page=pi” in the [[params]] field. (You use “&” to add additional parameters.) You use “page” because you are trying to find information specific to the individual user (by searching in his/her personal page). You set page=pi because pi refers to collaborators and co-authors. 

The output will show the number of collaborators the individual has worked with (“count”). (In this case, it is 1.) data shows the name, nsf_id and affiliation information (inst) for each collaborator.
