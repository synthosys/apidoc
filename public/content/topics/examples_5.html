Try It! http://nsf.api.smetrics.org/v1/topic?org=CHE&year=2010&t1=39,397,328&status=decline

Since the API includes private proposal data, proposals can have "award",  "decline" or "propose" status. 

To look for declined proposals in the same org with the same topic numbers as above, just attach “status=decline” to the end of the query using “&”. (You can have a similar query for "award"  or "propose" as well.)

The output includes additional request metadata, returning the request duration, date and dollar amounts.
