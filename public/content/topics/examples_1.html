To find basic information on proposals with specific topics, first identify that you are looking for a “topic” in the [[object]] field. Limit the topic by [[params]] such as year (in this case, 2010) and topic numbers (in this example, we want only information about proposals with the most common topic, t1, of 39, 397 or 328). 

The output shows the number of proposals that match the parameter specifications (“count”, in this case it is 194). It shows unique identifiers/codes for the proposal, organization affiliations, and various status updates, including “code” which shows whether the proposal is an “award”. topic shows the most common topic id (the first match shows 39, 397, or 328) for each particular proposal.
